# 🍓 SUPER BALDO – The Kingdom of Berries

A retro-inspired 2D platform game built entirely with **HTML5 Canvas**, **CSS**, and **vanilla JavaScript**.

Jump, collect berries, defeat enemies, unlock power-ups, and challenge the evil **King Melmone** in a classic arcade-style adventure.

---

## 🎮 Features

- 🕹️ Retro platformer gameplay
- 🍓 Collect berries and coins
- ⚡ Multiple power-ups
  - Giant Berry
  - Fire
  - Double Jump
  - Shield
  - Star Mode
- 👾 Different enemy types
- 👑 Boss battle
- ❤️ Lives and score system
- 🎵 Retro-style synthesized sound effects and music
- 📱 Touch controls for mobile devices
- 💻 No external libraries or frameworks
